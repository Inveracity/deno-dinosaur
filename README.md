# Vite + Deno + React

## Running

You need to have Deno v1.28.0 or later intalled to run this repo.

```sh
deno --version
# deno 1.28.3 (release, x86_64-unknown-linux-gnu)
# v8 10.9.194.5
# typescript 4.8.3
```

Run frontend:

```sh
deno task dev
```

Run API

```sh
deno run --allow-env --allow-net api/main.ts
```
